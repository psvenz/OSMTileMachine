Required tools and location:

1. maperitive		Place in tools/maperitive folder. 
					Download from http://maperitive.net

2. wget.exe			Place in path, for example c:\windows 
					Download from http://gnuwin32.sourceforge.net/packages/wget.htm

3. osmconvert.exe	place in path, for example c:\windows
					Download windows binary from http://wiki.openstreetmap.org/wiki/Osmconvert

4. osmupdate.exe	RENAME TO osmu.exe or osmup.exe and then place in path, for example c:\windows
					Download from http://wiki.openstreetmap.org/wiki/Osmupdate

5. Microsoft .NET 	Maperitive will need .NET version 4.
					Download via Windows Update, or from microsoft.com
					
6. Java				Java SE JRE version 8 needed.
					Download Java from www.oracle.com
