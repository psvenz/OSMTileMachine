osmTileMachine -operatingmode renderarea -area dalarna -debug
	Renders a map of dalarna. 


osmTileMachine -operatingmode updateplanet
	Runs osmupdate on the planet.pbf file. Downloads the planet.pbf first if missing.

osmTileMachine -operatingmode verifyplanet

osmTileMachine -operatingmode forceplanetdownload
	
	
	osmupdate.exe might have to be renamed osmu.exe or osmup.exe and placed in PATH. osmTileMachine can detect all three filenames. (windows 7 has some security warning issues with files ending with "XXXXXupdate.exe"
	osmconvert needs to be in PATH.
	wget needs to be in PATH.
