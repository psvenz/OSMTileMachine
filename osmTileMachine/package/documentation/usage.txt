Example usage:
osmTileMachine -download -update -renderarea Falun -debug
	Renders a map of dalarna. 

Parameter description: 

		-download
			Runs WGET and downloads the planet.pbf file. 
		
		-update
			Updating planet.pbf and produces planet_updated.pbf with the help from osmupdate tool.
		
		-renderarea yyyyy
			Splits planet_updated.pbf into smaller and smaller data files until each file could fit into memory of the render tool, then 
			feeds the datafiles to the render tool. (maperitive)

			Valid area sets the area to cover:
			Niemisel
			Falun
			Dalarna
			Scandinavia
			Niemisel
			Vänern
		
		-debug will give a verbose output in console

		-firstaction nn
			Will skip early actions and start with the specified action (resume previous operation, or for debug purposes)
		
		-maxzoom
			Max render depth. Recommended: 13 to 16.
